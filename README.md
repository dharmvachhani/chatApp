# chatApp 
### (Desktop Version)

ChatApp is a Multi user chatting app like Whatsapp web

# Quick start
Several quick start options are available:
* Clone the repo: git clone https://github.com/dharmvachhani/chatApp.git

* Install with npm: 
  `npm install`
  
* Start 
  `npm start`
